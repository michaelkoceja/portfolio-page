Alex Honnold

This site is a tribute page about a free solo climber named Alex Honnold. I built this page using Sass and Neat Grid. The layout of the sight itself is pretty simplictic. The simplicity of the site along with its grey color scale, give the site a nice touch. Check it out!

Pokemon Game

During our intro to programming class, I was part of a team that wanted to recreate a pokemon game. We wanted to replicate the classic gameboy style. For this game, we used Javascript to create the actual game where the user fights the pokemon. CSS was then used to style the page to make it look like a gameboy. Try the game out for yourself! Be sure to turn your volume on!
